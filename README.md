# Vanilla Firearms ++

## Important information:
### ONLY VISUAL:
This is a Project Zomboid Mod made to Improve visual and audio effect on the weapons of the B42.
This mod will never modify Weapons balance and gameplay on the Vanilla Weapons. 

### Realism:
This mod aim to bring realistic models, textures, animations, audios, names. Based on what we supposed to be there real counterparts.

## How to use it:
If you want to install the mod, please subscribe to it here:
https://steamcommunity.com/sharedfiles/filedetails/?id=3722358629

## How to contribute:
1. Clone the repository in your 
2. Create a fork of the project
3. Clone the fork repository in your `%userprofile%/Zomboid/Workshop/` Folder.
2. Modify the code, commit on your side, then make a pull request. [Github Guide: How To Pull Request from a Fork](https://docs.github.com/en/pull-requests/collaborating-with-pull-requests/proposing-changes-to-your-work-with-pull-requests/creating-a-pull-request-from-a-fork)
3. Wait me to review it, than, if it follow the guidelines and is a good addition to the mod, it will be added.


## Supported version:
✅ B42.17 or Higher
❌ previous B42 or B41
